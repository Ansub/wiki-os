# Getting Started

1. Set `WIKI_ROOT` to this sample vault or your own Obsidian folder.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the local URL the app prints.

## Tips

- Use article pages for long-form notes.
- Use tags to shape the homepage.
- Keep links between related notes.
